# Changelog

## [0.1.0] - Unreleased

### Added
- Initial release of Kizuna — self-hosted Discord alternative
- Real-time text chat with channels, categories, and direct messages
- Voice channels via WebRTC with mediasoup SFU
- Screen sharing support
- Custom roles and permissions system
- End-to-end encryption for direct messages
- File uploads and attachments
- GIF and sticker support with custom sticker packs
- Full-text message search
- Proof-of-Work registration protection
- Tauri v2 desktop application (Windows + Linux)
- Docker deployment with Caddy reverse proxy and automatic HTTPS
- UPnP NAT traversal support
